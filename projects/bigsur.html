Design Concept/Intent

When one types in “Big Sur” on a search engine, the Big Sur Chamber of Commerce website is the first item to appear. As an informational website, it is intended for tourists of short-term stays mostly, as well as any visitor about the famous area. I did some assessments on the website pros and cons and decided to redesign the whole website structure and user interface.

I want to make people feel happy and relaxed while browsing the website as well as showing them more beautiful pictures about Big Sur, by making the website look more appealing, modern and simple. The navigation system should be clearer and well-arranged for users to easily access information. I wanted it to be clean, flat, and focused.

Logo & Type Font

The original website, to be more precisely, the organization(Big Sur Chamber of Commerce) doesn’t have a logo and the way he present himself on the website is vague and faded in the header image and not so unique, so I tried to use type font to make it unique and outstanding, to tell people where the website comes from.

The type font that I choose is called Herculanum. It gives me a feeling of breezy and rocky, which I think is perfectly for Big Sur. For the rest of the content and navigation, I tried to use only one type font but with several varieties to present a type font hierarchy.
