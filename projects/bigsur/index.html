Big Sur Chamber of Commerce

Crafting relaxing getaways

Project Overview

When one types in “Big Sur” in a search engine, the Big Sur Chamber of Commerce website is the first item to appear. As an informational website, it is intended for tourists of short-term stays mostly, as well as any visitor from around the world that wants to know about the famous area. This project was to do assessments for the website and redesign the website structure and user interface.

My task

 In order to make people feel happy and relaxed while browsing the website, it needs to have a more appealing, modern and simple look. The navigation and information system should be focused, clearer and well-arranged for users to easily access information.
